# 🔍 ESG-LENS
### ESG 리스크를 통해 주식 시장을 바라보는 렌즈

> 기후 데이터 · KoBERT 감성 분석 · DART 공시 · LSTM 딥러닝 · Streamlit 대시보드

---

## 📌 프로젝트 개요

기존 주가 예측 모델은 과거 주가 수치만 활용합니다.
**ESG-LENS**는 기후 이상 현상과 뉴스 감성을 분석해 환경 리스크를 수치화하고,
이를 LSTM 딥러닝 모델에 융합하여 주가 방향을 예측하는 빅데이터 시스템입니다.

---

## 🧠 ESG 리스크 점수 산출

환경리스크 점수 = 뉴스 감성 점수 x 60% + 기후 이상 지수 x 40%

## 🤖 LSTM 예측 모델

| 모델 | 정확도 |
|------|--------|
| 주가만 사용 | 48.0% |
| 주가 + 기후 + ESG 점수 | **53.6%** (+5.6%p) |

- 입력: 30일 시퀀스 × 12개 피처
- 학습 데이터: 코스피 200 전체 224,950개 시퀀스
